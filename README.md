### **Projects & Experiences**

Here are some of the key machine learning projects I have worked on, focusing on real-world data analysis, predictive modeling, and customer behavior insights.

1. **Customer Segmentation using K-Means Clustering**
   - Developed a machine learning model to segment retail customers based on **Annual Income** and **Spending Score**.
   - Applied the **Elbow Method** to determine the optimal number of clusters and implemented K-Means clustering to identify different customer groups.
   - Built an **interactive dashboard with Flask backend and web interface** to visualize clusters and predict customer segments.
   - Tools: Python, Pandas, Scikit-learn, Flask, Matplotlib, Seaborn, HTML, CSS, JavaScript.

2. **Heart Disease Prediction (Machine Learning Web App)**
   - Built a machine learning model to predict the likelihood of heart disease based on patient health parameters.
   - Trained models using **Decision Tree and Random Forest algorithms**, improving prediction accuracy through model evaluation and tuning.
   - Integrated the trained model into a **Flask-based web application** with a simple HTML interface for real-time prediction.
   - Tools: Python, Flask, Scikit-learn, Pandas, HTML.

3. **Credit Card Fraud Detection**
   - Developed a machine learning model to detect fraudulent credit card transactions from a large dataset of financial records.
   - Addressed **class imbalance in fraud data** and applied machine learning techniques for better fraud detection performance.
   - Evaluated model performance using **precision, recall, confusion matrix, and precision–recall analysis** to ensure reliable detection.
   - Tools: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn.
